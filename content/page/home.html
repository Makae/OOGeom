Projektarbeit zur Objektorientierten Geometrie

Diese Seite entstand im Laufe des Modules "Objektorientierte Geometrie" der Berner Fachhochschule. Innerhalb des Grundinhaltes des Modules werden verschiedene Methoden besprochen wie Transformationen im n-dimensionalen Raum durchgeführt werden können. Der erste Teil des Modules besteht aus einem Labor in welchem die Studenten selber die vorgestellten Methoden anwenden sollen.
Zu diesem Labor wurde diese Webseite erstellt, welche die Grundinhalte und Aufgaben anschaulich und interaktiv präsentiert

Features

Folgende Features wurden für die Seite im Rahmen dieses Projektes umgesetzt

  • Interaktive 3D-Container für Codebeispiele
  • Annotierbare Codebeispiele für Parameteranpassungen der Scripts
  • Rekursive Codeansicht für die interessanten Codebeispiele
  • Networking-Klassen für das laden von inhalt
  • Dynamisches Contentloading
  • Ergänzung Matrix2-Klasse mit den wichtigsten Funktionen
  • Quaternion & DualQuaternion-Klassen
  • Code-Highlightung
  • Codebeispiele für die Laboraufgaben

Zum Code

Bis auf den libs-Ordner und alle extern inkludierten Javascript und CSS Dateien wurde der gesamte Inhalt während des Modules erstellt. Das GitHub-Projekt der Seite kann unter: https://github.com/Makae/OOGeom eingesehen werden.

Eingesetzte Libraries / Frameworks

  • ThreeJS : Darstellung der Aufgaben im 3D-Raum
  • Twitter Bootstrap : CSS-Framework für das Layout
  • HighlightJS : Code-Hilighting der Beispiele
  • jQuery : EventHandling in der ThreeJS-Facade
