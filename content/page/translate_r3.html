Translation mit Dualen Quaternion

Ein normales Quaternion allein kann keine Translation abbilden, dazu müssen Duale Quaternionen verwendet werden. Die Translation wird dabei im dualen Teil gespeichert.

𝔻ℍ = r + 𝜀d
while r, d ∈ ℚ   and  t is the desired translation
r = (1, 0, 0, 0)
d = (0,  tx/2, ty/2, tz/2)
